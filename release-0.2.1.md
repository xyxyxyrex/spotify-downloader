# SpotDL GUI v0.2.1 Release Notes

SpotDL GUI v0.2.1 is a feature and maintenance update to the desktop downloader.

Below is a detailed breakdown of all the changes compared to v0.2.0.

---

## New Features & Enhancements

* **Downloads Storage Details**: Added a display showing the total disk space occupied by all offline songs in the downloads directory.
* **Individual File Sizes**: The saved library list now displays the exact file size for each downloaded song next to its filename.
* **Open Downloads Directory**: Added a button that opens the downloads directory in the host operating system's native file explorer (explorer on Windows, Finder on macOS, xdg-open on Linux).

---

## Comparison Summary (v0.2.0 vs v0.2.1)

| Feature / Metric | v0.2.0 | v0.2.1 |
| :--- | :--- | :--- |
| **Status** | Released | Released |
| **Storage Tracking** | Not visible in app | Shows total downloads folder disk usage |
| **File Sizes Display** | Not shown | Individual sizes in table list |
| **Downloads Folder Shortcut** | Manual navigation | Dedicated button to open folder natively |
