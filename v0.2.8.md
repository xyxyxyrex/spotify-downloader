# SpotDL GUI v0.2.8 Changelog

## Synced Lyrics Integration
* Added synced lyrics retrieval by integrating the LRCLIB API (`lrclib.net`).
* Implemented multi-stage search fallback:
  1. Exact metadata matching via the `/api/get` endpoint using the exact artist and track name to prevent false matches.
  2. Broad query search fallback via the `/api/search` endpoint if the exact match fails.
  3. Plain lyrics retrieval fallback from `lyrics.ovh` if LRCLIB returns no result.
* Supports synchronized karaoke-style highlighting and interactive playback control (click a lyric line to seek to that timestamp).

## UI/UX Enhancements
* Added a dedicated full-screen lyrics panel with blurred album art background.
* Implemented automatic scroll centering to focus on the active lyric line during playback and seeks.
* Fixed text visibility/contrast issues in light theme mode.
* Replaced standard scrollbars with a custom, minimal scrollbar that hides when inactive.

## Bug Fixes
* Resolved an issue where seeking via lyrics before media caching completed caused playback to restart from the beginning.
* Fixed incorrect lyric matching for tracks with overlapping titles by enforcing exact metadata queries prior to broad search.
