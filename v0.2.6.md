# SpotDL GUI Release v0.2.6

## Features

### Plugin Marketplace & Management
* **Marketplace Tab**: Separated installed plugins from the plugin discovery catalog.
* **Remote Registry Integration**: Plugins are dynamically fetched from the [spoti-tauri-plugin-marketplace](https://github.com/xyxyxyrex/spoti-tauri-plugin-marketplace) repository and loaded via ES6 dynamic imports.
* **Uninstall Actions**: Added an uninstall button to remove active plugins from the user interface.
* **Cleaned UI**: Removed the "Load Local Plugin" option.

### Bundled Marketplace Plugins
* **Receiptify**: Renders a receipt-style visualization of listening history and play counts.
* **History Stats**: Displays play history reports analyzed from the local SQLite database.

### Debugging Capabilities
* **DevTools Integration**: Enabled Chrome/Safari DevTools via Tauri's `devtools` feature in development-configured environments.

### Bug Fixes
* **Playlist Query Resolution**: Fixed a crash in `spotify_query.py` when processing Spotify playlists by removing redundant initialization checks in `init_spotify()`.

## Compilation

### Prerequisites
Install Node.js dependencies:
```bash
npm install
```

### Run in Development Mode
```bash
npm run tauri dev
```

### Build Production Artifacts
```bash
npm run tauri build
```
